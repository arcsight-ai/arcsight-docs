# ArcSight v1 Scope (Locked)

## Scope

- ArcSight v1 addresses Structural Certainty only.

## Primary Question Answered

- “Can I safely change or remove this?”

## In Scope

- Structural dependency analysis
- Cycles and non-local change impact
- Signals that demonstrate change impact is not locally reasoned about

## Explicitly Out of Scope (Deferred)

- Semantic contracts
- Behavioral meaning or intent
- Temporal readiness or timing
- Confidence scoring
- Usage inference
- Ecosystem signals
- Risk explanations
- Recommendations or advice

## Non-Goals

- Broad architectural assessment
- Code quality evaluation
- Refactoring guidance
- Developer education
